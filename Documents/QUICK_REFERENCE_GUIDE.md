# Windows SIP Phone - Quick Reference Guide

## 🚀 Getting Started

### 1. Initial Setup
1. **Launch Application**: Double-click `WindowsSipPhone.exe`
2. **Configure SIP Settings**: Go to Settings tab
   - Server: Your SIP server IP/hostname
   - Username: Your SIP extension
   - Password: Your SIP password
   - Port: Usually 5060
3. **Register**: Click "Register" button
4. **Verify**: Status bar shows "🟢 Ready - SIP Registered"

### 2. Making Calls
1. **Go to Dialer Tab**
2. **Enter Number**: Use number pad or type directly
3. **Call**: Click "📞 Call" button
4. **Talk**: Audio starts automatically when connected
5. **Hangup**: Click "📵 Hangup" when done

### 3. Receiving Calls
1. **Stay Registered**: Keep application running
2. **Incoming Call**: Notification window appears automatically
3. **Answer**: Click "Accept" button
4. **Decline**: Click "Decline" button if unwanted

## 🎛️ Audio Controls

### During Active Calls
- **Mute/Unmute**: Click audio button (🔊/🔇)
- **Volume**: Use slider in volume control section
- **Quality**: Automatic G.711 codec for clear audio

### Audio Indicators
- 🔊 **Audio**: Microphone active
- 🔇 **Unmute**: Microphone muted
- Volume slider shows current level (0-100%)

## 📞 Call Management

### Call Status Display
- **"Calling [number]..."**: Outgoing call in progress
- **"Connected to [number]"**: Active call established
- **Duration Timer**: Shows elapsed call time (MM:SS)

### Call History
- **View**: Right panel of Dialer tab
- **Filter**: Use buttons (All, Outgoing, Incoming, Missed)
- **Redial**: Select call and click "📞 Redial"
- **Clear**: Click "🗑️ Clear History" to remove all

## 🔧 Troubleshooting

### Common Issues & Solutions

| Problem | Solution |
|---------|----------|
| Registration fails | Check server IP, username, password, and port |
| No audio | Verify microphone/speaker permissions in Windows |
| Can't make calls | Ensure registration is successful first |
| Poor audio quality | Check network connection and audio device settings |
| Incoming calls not received | Verify registration and firewall settings |

### Status Indicators
- 🟢 **Green**: Registered and ready
- 🟡 **Yellow**: Connecting/registering
- 🔴 **Red**: Not connected
- 📞 **Phone**: Active call in progress

## 🎯 Best Practices

### Audio Quality
- Use headset for best experience
- Keep volume at 50-80% for optimal quality
- Mute when not speaking in group scenarios
- Test audio before important calls

### Call Management
- End calls properly using hangup button
- Review call history regularly
- Use redial for frequently called numbers
- Keep application running to receive calls

### System Performance
- Close unnecessary applications during calls
- Ensure stable internet connection
- Keep Windows audio drivers updated
- Monitor application for any errors

## 📞 Emergency Information

### If Application Becomes Unresponsive
1. Close application using Task Manager
2. Restart application
3. Re-register with SIP server
4. Resume normal operation

### If Audio Problems Persist
1. Check Windows audio settings
2. Restart application
3. Test with different audio device
4. Verify network connection
